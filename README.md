# 2D Graphics Editor
A menu-driven console application built in C that allows users to create, view, and modify geometric shapes on an interactive character-based matrix canvas.

## Features
Draw Shapes:  Supports rendering Rectangles, Lines, Triangles, and Circles.                                

Interactive Controls:Clear canvas functionality and shape deletion.
Custom Character Rendering: Dynamically plots shapes using custom characters like * on an adjustable grid.

## How to Run
1.Open the project in VS Code.
2.Compile and run main.c using a C compiler (GCC).2.
